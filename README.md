# Fuji Look Assistant

Fuji Look Assistant is a Streamlit app that analyzes a reference image and suggests a Fujifilm-style JPEG recipe direction for X-Trans IV and X-Trans V cameras.

It is designed as a creative assistant, not an exact film recipe matcher. The app gives a starting point based on the look, feel, color, tone, and mood of the uploaded image.

## Features

- Upload a reference image
- Analyze color, contrast, warmth, saturation, and mood
- Suggest a Fujifilm film simulation direction
- Generate sensor-safe recipe settings
- Supports X-Trans IV and X-Trans V
- Shows top alternative looks
- Export recipe as TXT
- Export full analysis as JSON
- Dark professional UI

## Supported Sensor Generations

- X-Trans IV
- X-Trans V
